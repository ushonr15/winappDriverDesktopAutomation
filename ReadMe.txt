One time Setup
	1. Download and install latest windows Application Driver 64 installer from https://github.com/microsoft/WinAppDriver/releases/tag/v1.2.99
	2. Enable Developers Mode from Windows settings
	3. Make sure inspect.exe is insatlled on your machine unnder C:\Program Files (x86)\Windows Kits\10\bin\10.0.xxxx\x64\inspect.exe
	

To get better execution performace please close all open applications
